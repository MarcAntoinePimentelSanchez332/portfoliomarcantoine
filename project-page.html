Key Features

Interactive User Interface:
Built using PyQt5, the application offers a user-friendly interface for inputting parameters such as diameter, pitch, length, and tensile strength, along with process and type details.

Advanced Machine Learning Model :
Utilizes a trained MLP (Multi-Layer Perceptron) Regressor for precise force predictions. The model is dynamically loaded from saved files, ensuring up-to-date predictions.

Data Normalization and Encoding : Incorporates MinMaxScaler for normalizing input data and LabelEncoder for handling categorical data, improving the model's performance.

Excel Integration : Features capabilities to read and write data to Excel files, facilitating easy data management and storage.

Real-Time Predictions and Data Management :
Allows users to perform predictions, manually adjust them, and manage data directly within the application.

Customized Data Visualization : Offers an integrated table view to display predictions and associated data, with color-coded feedback on prediction accuracy.

Technical Stack

Languages and Frameworks :
Python, PyQt5, Anaconda

Libraries :
Scikit-Learn, Pandas, OpenPyXL

Tools : cProfile for performance profiling




Challenges And Solutions

- Ensuring compatibility with different versions of Scikit-Learn and handling potential issues arising from model version mismatches.

- Implementing a robust and interactive GUI that seamlessly integrates with the backend machine learning model.

- Managing data efficiently between the application and Excel files, including real-time updates and search functionality.



**Outcome:**

The application successfully bridges the gap between complex machine learning algorithms and practical industrial applications, providing a valuable tool for engineers and technicians in the manufacturing industry. It demonstrates effective use of Python for developing applications that are both technically sophisticated and user-friendly.
